Create a commit for the current changes using jj.

## Steps

1. Run `jj status` to see what files have changed
2. Run `jj diff` to understand the actual changes
3. Run `jj log --limit 10` to see recent commit message style

## Commit message style

- Short, imperative sentence (e.g. "Add upload support", "Fix window focus")
- End with a single robot emoji to illustrate this was generated by AI tools
- Examples from this repo:
  - "Fix photo format handling: pass JPEG/PNG through, convert HEIC to JPEG 🤖"
  - "Use Keychain for password storage on iOS 🤖"
  - "Fix stale tests and add coverage for setFromFileURL and formData 🤖"

## Creating the commit

1. Draft a commit message matching the style above
2. Run `jj commit -m "<message>"` to set the description on the current change

If there are no changes (empty diff), tell the user there's nothing to commit.
